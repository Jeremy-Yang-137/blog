# 3D Gaussian Splatting

一种用高斯球（很多个椭圆）来表现3D场景的方式。通过大量不同角度拍摄的图片来训练。

[graphdeco-inria/gaussian-splatting: Original reference implementation of “3D Gaussian Splatting for Real-Time Radiance Field Rendering”](https://github.com/graphdeco-inria/gaussian-splatting) (Github)

[Bilibili视频教程](https://www.bilibili.com/video/BV1Gu3uztECe)

需要colmap来计算相机位置。

1. 把素材图片放到/data/input里
2. 运行python convert.py -s data
3. 运行python train.py -s data（可以打开Viewer实时查看）
4. 生成的模型在/output里，.ply格式

前置：用ffmpeg切片视频+用ImageMagick缩小图片

```bash
ffmpeg -i input.mp4 -vf "setpts=0.2*PTS" %4d.jpg
python convert.py -s data --resize
```
生成的ply文件可以在[SuperSplat Editor](https://superspl.at/editor)中查看。

# ml-sharp

通过一张图片生成3D Gaussian Splatting场景（.ply格式）。

[apple/ml-sharp: Sharp Monocular View Synthesis in Less Than a Second](https://github.com/apple/ml-sharp) (Github)

```bash
conda activate sharp
cd C:\Users\Yang\Desktop\Coding\Research\ml-sharp
sharp predict -i INPUT -o OUTPUT
```
原始图片在INPUT里，生成的.ply文件在OUTPUT里。
