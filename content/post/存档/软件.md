# VS Code

配置文件（profile）功能非常好用，可以让不同的配置有各自独立的插件和设置。

打开内置命令行的快捷键是Ctrl+`。

鼠标中键拖动可以矩形选择，Alt+左键可以设置多重光标，Ctrl+/可以快速注释。

### VS Code主题开发

- 通过npm install -g yo generator-code以及yo code选择创建主题
- 用VS Code打开文件夹，修改theme.json中的颜色值，可以按F5打开调试窗口看效果，用Developer: Inspect TM Scopes来查看编辑器中的语法类型
- 最后用vsce来打包以安装
