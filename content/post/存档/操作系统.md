# Windows

我一般习惯所有应用都在开始菜单启动，桌面上只放文件和回收站。左右边各几列，临时文件浮在中间。

不安装到系统的程序我统一放在C:\Software下，然后添加到path。

如果需要使用管理员身份的命令，要在开始菜单右击终端->以管理员身份运行。

两个小工具：

- Watt Toolkit：可以进行Github和Steam的加速
- Powertoys：有很多小工具，如取色器、窗口固定、快速搜索等
可以在语言设置里安装一个纯英文的输入法，通过win+空格与原来的中英文输入法切换，在写纯英文代码或打游戏时可以打开。

可以在注册表里把Edge的BrowserSignin设置为0，防止它总是莫名其妙地登录然后同步数据。

### 重装
需要下载Windows官方安装镜像，用软碟通刷到U盘里，bios设置U盘启动，然后插上U盘开机安装。

Win10和Win11都可以，但Win7可能由于硬件不支持，在我之前的笔记本上安装不了。

# Linux

可以通过三种方式运行：

- WSL
- 虚拟机（Virtual Box、VMware等）
- 直接装到电脑里（单系统/双系统）
可以通过ssh远程连接，甚至在VS Code里可以直接打开文件夹，很方便。

值得一提的是，Linux采用inode管理文件和目录，很多文件系统的特性都与它有关。

### 重装

和Windows基本一样。不过Linux的安装过程普遍比较复杂，Ubuntu算是简单的。

需注意Linux不一定自带硬件的驱动。如果用的是很新的电脑，最好下载最新的系统安装（我就遇到intel过网卡驱动的问题）。

可以通过安装grub引导，与Windows Boot Loader共存，实现双系统。不过其实不太方便。

# macOS

应用是打包到.app文件里，通过打开dmg磁盘映像直接拖到Applications文件夹里安装的。很神奇。

不过要注意架构的问题，M1往后的芯片都是arm64架构。
